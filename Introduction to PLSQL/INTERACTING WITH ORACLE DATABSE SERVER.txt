INTERACTING WITH ORACLE DATABSE SERVER

Q1) Write a program that displays the Netsal (Sal+comm) of an employee entered at run time from EMP table. (Use NVL Function).

SOLUTION) 
DECLARE
    v_empno EMP.EMPNO%TYPE := &employee_id; -- Accepts input at run time
    v_netsal NUMBER(10,2);
BEGIN
    SELECT sal + NVL(comm, 0) 
    INTO v_netsal
    FROM EMP
    WHERE EMPNO = v_empno;

    DBMS_OUTPUT.PUT_LINE('The Net Salary for Employee ' || v_empno || ' is: ' || v_netsal);
EXCEPTION
    WHEN NO_DATA_FOUND THEN
        DBMS_OUTPUT.PUT_LINE('Error: Employee ID ' || v_empno || ' does not exist.');
    WHEN OTHERS THEN
        DBMS_OUTPUT.PUT_LINE('An unexpected error occurred.');
END;
/
